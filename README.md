# Telegram Support Bot (aiogram)

Бот поддержки для Telegram на базе aiogram 3.x

## Описание

Простой бот поддержки, который:
- Принимает обращения от пользователей через команду `/start`
- Пересылает сообщения администратору
- Позволяет администратору отвечать пользователям от имени бота

## Установка

1. Клонируйте репозиторий или скачайте файлы

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните `.env` файл:
   - `BOT_TOKEN` - токен вашего бота от @BotFather
   - `ADMIN_ID` - ваш Telegram ID или список ID через запятую (можно узнать у @userinfobot)
     - Для одного администратора: `ADMIN_ID=123456789`
     - Для нескольких: `ADMIN_ID=123456789,987654321,111222333`

## Запуск

```bash
python bot.py
```

## Как использовать

1. Пользователь отправляет команду `/start` боту
2. Бот отправляет приветственное сообщение
3. Пользователь пишет свой вопрос
4. Бот подтверждает получение сообщения (только после первого сообщения)
5. Сообщение отправляется всем администраторам с информацией о пользователе
6. Любой администратор может ответить на сообщение (reply)
7. Ответ отправляется пользователю от имени бота

## Особенности

- Поддержка нескольких администраторов (указывайте ID через запятую)
- Автоматическая отправка сообщений всем администраторам
- Сохранение связи между сообщениями пользователей и администраторов
- Поддержка текстовых сообщений, фото, документов, видео, аудио и других типов медиа
- Подтверждение получения сообщения только после первого сообщения пользователя
- Логирование всех операций

## Настройка бота

Для настройки бота через @BotFather:
- Название: `Support`
- Username: `supportbet_bot`
- Описание: можно добавить описание поддержки

## Требования

- Python 3.8+
- aiogram 3.4.1
- python-dotenv 1.0.0
